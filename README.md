# Library-Management-System-
## Overview

The Library Management System is a console-based Python application developed using Object-Oriented Programming (OOP). It allows users to manage library books efficiently by adding, viewing, searching, issuing, and returning books. The application uses JSON file handling to store data permanently.

---
##  Features

- Add New Book
- View All Books
- Search Book by ID or Title
- Issue Book to Student
- Return Book
- Automatic Data Storage using JSON
- Duplicate Book ID Validation
- Quantity Validation
- Menu-Driven Console Application

---

##  Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- JSON
- File Handling
- OS Module
